# Contributing

## Setup

```bash
npm install
npm run build
```

## Running Tests

```bash
npm test
npm run test:watch
```

## Typecheck

```bash
npm run typecheck
```

## PR Guidelines

1. Fork the repo
2. Create a feature branch
3. Add tests for new functionality
4. Ensure all tests pass and typecheck passes
5. Update documentation if needed
6. Submit a PR
